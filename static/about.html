Architecture

The five-agent pipeline

Every query flows through a deterministic graph of specialised agents, each with a single responsibility.

01

Orchestrator

Decomposes your question into 3–5 specific, independently searchable subtasks and writes a research strategy.

02

Research Agent(s)

Uses Tavily's web search API to gather sources for each subtask. In parallel mode, one agent per subtask runs concurrently via LangGraph's Send() API.

03

Analysis Agent

Synthesises all gathered sources into a coherent analysis — mapping key findings, tradeoffs, areas of consensus, and confidence levels.

04

Critique Agent

Reviews the analysis for unsupported claims, missing perspectives, bias, and blind spots. Flags issues for the synthesiser to address.

05

Synthesiser

Writes the final publication-ready report with an executive summary, numbered findings with citations, tradeoffs, limitations, and a conclusion.

Tech Stack

Built on open tools

LLM runtime
Groq + Llama 3.3 70B
Ultra-fast inference via Groq's LPU hardware. Llama 3.3 70B for high-quality reasoning at speed.
Orchestration
LangGraph
Typed stateful agent graphs with built-in support for parallel fan-out via the Send() API.
Web search
Tavily
Purpose-built search API for AI agents — returns structured results with URLs, titles, and content snippets.
API server
FastAPI + SSE
Async Python server with Server-Sent Events for real-time streaming of agent progress to the browser.
Setup

Run it yourself

Install dependencies, add your API keys, and launch the server.

# 1. Install dependencies
pip install -r requirements.txt

# 2. Copy and fill in API keys
cp .env.example .env

# 3. Start the web server
uvicorn api:app --reload --port 8000

# 4. Open in browser
http://localhost:8000

You need a free Groq API key from console.groq.com and a Tavily API key from tavily.com (1,000 free searches/month).
